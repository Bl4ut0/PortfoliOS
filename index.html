How to Configure Google OAuth

                                            
To connect your personal Google Drive, you need to create a free Web OAuth Client ID on Google Cloud Console:

                                            
    
                                                
  • Go to the Google Cloud Console.
    • 
                                                
  • Select or create a project, then search for and enable the Google Drive API.
    • 
                                                
  • Set up your OAuth Consent Screen and add the .../auth/drive.file scope.
    • 
                                                
  • Navigate to Credentials > Create Credentials > OAuth client ID.
    • 
                                                
  • Set the application type to Web application.
    • 
                                                
  • In Authorized JavaScript Origins, add:
                                                    
      
                                                        
    • https://os.bl4ut0.dev
      • 
                                                        
    • http://localhost:8005 (for local testing)
      • 
                                                    
    
                                                
    • 
                                                
  • Leave Authorized redirect URIs empty (the token client handles returns directly via JavaScript origin messaging).
    • 
                                                
  • Copy the generated Client ID and paste it into the field above.
